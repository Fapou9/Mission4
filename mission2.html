Mission 2


L'objectif de cette mission est de débuter l'exploitation de la base de données de l'entreprise KDou afin de permettre, à terme, la production d'analyses et d'indicateurs pertinents. L'utilisation du logiciel Low Code Access est prévue pour la réalisation des requêtes nécessaires.


Le premier indicateur intéressant est le "Quelles sont les produits de type viande livrés à Paris ?".
Cet indicateur est intéressant car nous pouvons voir quels genres de produit sont livré à Paris, ça peut également nous donner une idée sur les produits les plus populaire consommé à la capitale.


Résultat :



SQL :


SELECT DISTINCT Produit.NomProd, Client.VilleCli, Categorie.NomCateg
FROM Client INNER JOIN Commande
ON Client.CodeCli = Commande.CodeCli, Categorie INNER JOIN Produit ON Categorie.CodeCateg = Produit.CodeCateg
WHERE (((Client.VilleCli)="Paris") AND ((Categorie.NomCateg)="Viandes"));
